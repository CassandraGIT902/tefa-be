## Getting Started

First, install dependencies packages that we need:

```bash
npm install
```
And then create new db and migrate, seed data to db
db : db_tefa

```bash
npx prisma generate
npx prisma migrate dev
npm run seed

```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
